# restplanner
REST API for financial planner built around pymortgage

## Development

* grab pymortgage
* create virtual environment:

```bash
cd restplanner
virtenv venv
. venv/bin/activate
```

* install pymortgage into venv:

```bash
cd pymortgage
python setup.py install
```

* run dev server:

```bash
python mortgage_rest.py
```

* connect to localhost:5000 for testing
